# Stress Categories

Bacterial StressomeDB contains curated stress category labels organized into nine higher order groups.

| Higher order group | Example category labels |
|---|---|
| Oxidative stress | Oxidative stress |
| Metal homeostasis and resistance | Metal toxicity, copper resistance, arsenic resistance |
| Acid adaptation | Acid stress, acid resistance |
| Envelope stress | Envelope stress |
| Heat shock | Heat shock |
| Osmotic stress | Osmotic stress |
| Detoxification systems | DNA damage, DNA repair, detoxification related systems |
| Multidrug and biocide efflux | Multidrug stress, efflux pump, ABC transporter |
| Global stress regulation | Starvation, two component systems, unclassified broad stress responses |

The higher order grouping was introduced to support comparative analyses across bacterial taxa while preserving more detailed source category labels in the database.
